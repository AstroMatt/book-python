Constants
=========

Pi number
---------
.. code-block:: python

    import numpy as np


    np.pi
    # 3.1415926535897931

Euler number
------------
.. code-block:: python

    import numpy as np


    np.e
    # 2.7182818284590451

Not-a-Number
------------
.. code-block:: python

    import numpy as np


    np.nan
    # NaN

Infinity
--------
.. code-block:: python

    import numpy as np


    np.inf
    # inf
