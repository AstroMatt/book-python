*****************
DataFrame Slicing
*****************

.. code-block:: python

    import numpy as np
    import pandas as pd

    np.random.seed(0)


    values = np.random.randn(6, 4)
    columns = ['Morning', 'Noon', 'Evening', 'Midnight']
    indexes = pd.date_range('1970-01-01', periods=6)

    df = pd.DataFrame(values, index=indexes, columns=columns)
    #               Morning       Noon    Evening   Midnight
    # 1970-01-01   0.486726  -0.291364  -1.105248  -0.333574
    # 1970-01-02   0.301838  -0.603001   0.069894   0.309209
    # 1970-01-03  -0.424429   0.845898  -1.460294   0.109749
    # 1970-01-04   0.909958  -0.986246   0.122176   1.205697
    # 1970-01-05  -0.172540  -0.974159  -0.848519   1.691875
    # 1970-01-06   0.047059   0.359687   0.531386  -0.587663


Slicing by index
================

Single row
----------
.. code-block:: python

    df[1]
    # 1970-01-02   0.301838  -0.603001   0.069894   0.309209

Rows range
----------
.. code-block:: python

    df[1:3]
    # 1970-01-02   0.301838  -0.603001   0.069894   0.309209
    # 1970-01-03  -0.424429   0.845898  -1.460294   0.109749


Slicing by columns
==================

Single column
-------------
.. code-block:: python

    df.Morning
    # 1970-01-01   -0.438232
    # 1970-01-02   -1.798254
    # 1970-01-03   -0.802938

    # 1970-01-05    1.800466
    # 1970-01-06    0.141029
    # Freq: D, Name: Morning, dtype: float64

.. code-block:: python

    df['Morning']
    # 1970-01-01   -0.438232
    # 1970-01-02   -1.798254
    # 1970-01-03   -0.802938
    # 1970-01-04    0.820863

    df[['Morning', 'Evening']]
