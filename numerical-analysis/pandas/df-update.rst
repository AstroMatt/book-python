Fill NaN - Forward Fill
=======================
* Values from previous row
* ``ffill``: propagate last valid observation forward

.. code-block:: python

    import pandas as pd
    import numpy as np


    df = pd.DataFrame({
        'A': [1, 2, np.nan, np.nan, 3, np.nan, 4],
        'B': [1.1, 2.2, np.nan, np.nan, 3.3, np.nan, 4.4],
        'C': ['a', 'b', np.nan, np.nan, 'c', np.nan, 'd'],
        'D': [np.nan, np.nan, np.nan, np.nan, np.nan, np.nan, np.nan],
    })

    df
    #      A    B    C   D
    # 0  1.0  1.1    a NaN
    # 1  2.0  2.2    b NaN
    # 2  NaN  NaN  NaN NaN
    # 3  NaN  NaN  NaN NaN
    # 4  3.0  3.3    c NaN
    # 5  NaN  NaN  NaN NaN
    # 6  4.0  4.4    d NaN

    df.fillna(method='ffill')
    #      A    B  C   D
    # 0  1.0  1.1  a NaN
    # 1  2.0  2.2  b NaN
    # 2  2.0  2.2  b NaN
    # 3  2.0  2.2  b NaN
    # 4  3.0  3.3  c NaN
    # 5  3.0  3.3  c NaN
    # 6  4.0  4.4  d NaN
