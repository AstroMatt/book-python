Series Slicing
**************


Numeric Index
=============
.. code-block:: python

    import pandas as pd

    s = pd.Series([11, 22, 33, 44, 55])

    s
    # 0    11
    # 1    22
    # 2    33
    # 3    44
    # 4    55
    # dtype: int64

    s[:2]
    # 0    11
    # 1    22
    # dtype: int64

    s[2:]
    # 2    33
    # 3    44
    # 4    55
    # dtype: int64

    s[1:-2]
    # 1    22
    # 2    33
    # dtype: int64

    s[::2]
    # 0    11
    # 2    33
    # 4    55
    # dtype: int64

    s[1::2]
    # 1    22
    # 3    44
    # dtype: int64


String Index
============
* Using string index upper and lower bound are inclusive!
* String indexes has also numeric index underneath

.. code-block:: python

    import pandas as pd

    s = pd.Series(
        data = [11, 22, 33, 44, 55],
        index = ['a', 'b', 'c', 'd', 'e'])

    s
    # a    11
    # b    22
    # c    33
    # d    44
    # e    55
    # dtype: int64

    s['a':'b']
    # a    11
    # b    22
    # dtype: int64

    s['a':'d']
    # a    11
    # b    22
    # c    33
    # d    44
    # dtype: int64

    s['a':'d':2]
    # a    11
    # c    33
    # dtype: int64

    s['a':'d':'b']
    # TypeError: '>=' not supported between instances of 'str' and 'int'

    s['d':'a']
    # Series([], dtype: int64)

Date Index
==========
.. code-block:: python

    import pandas as pd

    s = pd.Series(
        data = [11, 22, 33, 44, 55],
        index = pd.date_range('1999-12-30', periods=5))

    s
    # 1999-12-30    11
    # 1999-12-31    22
    # 2000-01-01    33
    # 2000-01-02    44
    # 2000-01-03    55
    # Freq: D, dtype: int64

    s['2000-01-02':'2000-01-04']
    # 2000-01-02    44
    # 2000-01-03    55
    # Freq: D, dtype: int64

    s['2000-01-02':'2000-01-04':2]
    # 2000-01-02    44
    # Freq: 2D, dtype: int64

    s['2000-01-02':'2000-01-04':-1]
    # Series([], Freq: -1D, dtype: int64)

    s['2000-01-04':'2000-01-02':-1]
    # 2000-01-03    55
    # 2000-01-02    44
    # Freq: -1D, dtype: int64

    s['1999-12':'1999-12']
    # 1999-12-30    11
    # 1999-12-31    22
    # Freq: D, dtype: int64

    s['2000-01':'2000-01-05']
    # 2000-01-01    33
    # 2000-01-02    44
    # 2000-01-03    55
    # Freq: D, dtype: int64

    s[:'2000-01-05':2]
    # 1999-12-30    11
    # 2000-01-01    33
    # 2000-01-03    55
    # Freq: 2D, dtype: int64

    s[:'2000-01-03':-1]
    # 2000-01-03    55
    # Freq: -1D, dtype: int64

    import pandas as pd
